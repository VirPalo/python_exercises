# DISTANCIA ENTRE PUNTOS EN EL PLANO

## Descripción
Vamos a suponer que necesitas conocer la distancia entre dos puntos con coordenadas conocidas. Entonces, este proyecto te pide que ingreses las coordenadas una por una y realiza el cálculo de la distancia, siempre que se trate de un float, ya que hace una validación para verificar que los datos sean correctamente ingresados.

## Contenido
- Calculo distancia entre puntos  
- Validación de entrada de usuario  
- Código modularizado con funciones  
- Documentación con docstrings
